About This Project

A data visualization task developed for a position at 3ivis.

Purpose

This application is designed to display historical GDP data interactively, allowing users to analyze economic trends over time. The project demonstrates core skills in building RESTful APIs, frontend data visualization with D3.js, and secure authentication.

Features

  • Data Visualization: Interactive line charts rendered with D3.js
  • API Access: RESTful API with token-based authentication to retrieve chart data
  • Secure Authentication: Only authenticated users can access the chart page

Technical Stack

  • Django and Django REST Framework for backend
  • D3.js for interactive data visualization
  • Bootstrap for responsive and attractive design
